# Game Market — тестовая верстка

Десктопная главная страница магазина цифровых товаров по макету Figma. Реализована на чистых HTML, CSS и JavaScript без сборщика и фреймворков.

## Запуск

Откройте `index.html` напрямую или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Страница будет доступна по адресу `http://localhost:8000`.

## Интерактивы

- автоматическая баннер-карусель, стрелки и индикаторы;
- каталог по клику, закрытие повторным кликом, вне меню и по Escape;
- переключатель валюты в пополнении Steam;
- hover-состояния сервисов;
- hover-состояния товарных карточек.
